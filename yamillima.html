Yamil Lima - Desarrollador del Producto

Experiencia

A mis 11 años de edad aprendí HTML, CSS y JavaScript por internet, en forma autodidacta, desde el pc de mi familia. Posteriormente he aprendido Java y Python, teniendo hoy la capacidad de desarrollar aplicaciones móviles nativas Android y webs tanto en el frontend como en el backend con el framework Django basado en Python. Algunos de mis desarrollos destacados son Gugif.com, un marketplace para conectar tiendas de regalos y consumidores finales, construido con Django; y un prototipo inicial de AmbulanciApp en Android, que utiliza la API de Google Maps y busca conectar ciudadanos con ambulancias, IPS y aseguradoras.

Portafolio

Gugif.com - Marketplace para conectar tiendas de regalos y consumidores finales. Construido con el framework Django basado en Python.

AmbulanciApp - Repositorio del frontend del prototipo inicial de AmbulanciApp para el ciudadano o usuario final. Aplicación móvil construida en Android, para conectar ciudadanos con ambulancias, IPS y aseguradoras.
